Scrapyard

Firefox Add-On :: Chrome Extension

SEE ALSO: iShell

Description

Scrapyard is a bookmarking extension where you can store and organize bookmarks, page fragments, complete HTML pages and PDF documents, or take notes. It is intended as a modern alternative to the legacy ScrapBook add-on. All archived content is accessible on the Android platform if shared through a cloud service. It is possible to export and import bookmarks in JSON or org-mode formats. Scrapyard also supports the import of RDF archives of the legacy ScrapBook addon. See the online help for a more detailed description of the add-on features.

Key Features

  • Friendly user experience close to the original ScrapBook.
  • Browser bookmark integration.
  • Multiple bookmark shelves.
  • Ability to save page fragments and PDF documents.
  • Ability to automatically archive entire sites.
  • It is possible to attach text notes to any bookmark or archived page.
  • Cloud bookmarking.
  • TODO functionality.

Cloud Bookmarking

Scrapyard has built-in support of Dropbox and OneDrive cloud services and allows sharing bookmarks into the special Cloud shelf, which could be accessed across different browser instances. It is also possible to browse the content of this shelf and share links to it from mobile devices through the Scrapyard Android application. The application also could pick up the complete Scrapyard archive shared through one of the cloud services mentioned above.

Android application

Scrapyard for Android allows to share links and text to Scrapyard Cloud shelf from any app on mobile devices.

Limitations

  • Text shared from the Android platform does not include links or images.
  • The selected text is shared as notes without a source URL. Share text from the Pocket app or the recent versions of Chrome to save source URLs automatically.
