Scrapyard

Firefox Add-On :: Chrome Extension

SEE ALSO: iShell

Description

Scrapyard is a bookmarking extension where you can store and organize bookmarks, page fragments, or complete documents in unlimited quantities. It is intended as a modern alternative to the legacy ScrapBook add-on. By utilizing browser internal storage Scrapyard may operate without external tools and allows to import/export bookmarks in JSON and org-mode formats or synchronize them through a cloud service. It is also possible to access all saved content on mobile devices. See the online help for a more detailed description of the add-on features.

Key Features

  • Friendly user experience close to the original ScrapBook.
  • Firefox bookmark integration.
  • Multiple bookmark shelves.
  • Ability to save page fragments and PDF documents.
  • Ability to automatically archive entire sites.
  • It is possible to attach text notes to any bookmark or archived page.
  • Cloud bookmarking.
  • TODO functionality.
  • File-based synchronization.
  • Bookmarks synchronized through Dropbox or OneDrive are available on Android mobile devices through the Scrapyard Android application.
  • iShell integration.
  • Automation

Cloud Bookmarking

Scrapyard has built-in support of Dropbox and OneDrive cloud services and allows sharing bookmarks into the special Cloud shelf, which could be accessed across different browser instances. It is also possible to browse the content of this shelf and share links to it from mobile devices through the Scrapyard Android application. By using the Cloud shelf, you can access selected bookmarks or archived pages on all your devices.

The Sync option allows you to synchronize all your bookmarks across different browser instances and access them in the Scrapyard Android application. Please see the user manual for more details.

Android application

Scrapyard for Android allows to share links and text to Scrapyard Cloud shelf from any app on mobile devices.

Limitations

  • Text shared from the Android platform does not include links or images.
  • The selected text is shared as notes without a source URL. Share text from the Pocket app or the recent versions of Chrome to save source URLs automatically.
