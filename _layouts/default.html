Scrapyard

Firefox Add-On

SEE ALSO: UbiquityWE

Description

Scrapyard is a Firefox extension that resides in sidebar where you can store and organize bookmarks, page fragments or complete documents in unlimited quantities. By utilizing browser local storage Scrapyard does not require external tools to operate and allows to import/export data in JSON or org-mode formats.

Features

  • Firefox bookmark integration.
  • Multiple bookmark shelves.
  • Ability to save page fragments and entire PDF-documents.
  • Cloud bookmarking.
  • TODO functionality.
  • Import/export from org-mode files HTML-bookmarks.
  • It is possible to attach text notes to any bookmark or archived page.
  • Sharing to Pocket or Dropbox.
  • UbiquityWE integration.
  • Import of Scrapbook RDF archives.
  • No need to install any external tools.

Limitations

  • Because Scrapyard tries to pack all page fonts, images and styles into a single file to keep it in the database, it may perform substantially slow when archiving heavy pages.
  • Due to Firefox security restrictions all scripts are removed from the archived pages.
  • Resources in <embed> and <object> tags are not saved by default and currently not displayed due to Firefox security restrictions.

Android application

Scrapyard for Android allows to share links and text to Scrapyard Cloud shelf from any app on mobile devices.

Limitations

  • It is only possible to share plain text without links and images from Android platform.
  • Selected text is shared as notes without source URL. Share from Pocket app to save source URL automatically.
