Scrapyard

Install Firefox Add-On

SEE ALSO: iShell

Description

Scrapyard is a Firefox bookmarking extension that resides in sidebar where you can store and organize bookmarks, page fragments or complete documents in unlimited quantities. It is intended as a modern alternative to the legacy ScrapBook add-on. By utilizing browser local storage Scrapyard does not require external tools to operate and allows to import/export data in JSON or org-mode formats. See the online help for the more detailed description of the add-on features.

Features

  • User experience close to the original ScrapBook.
  • Firefox bookmark integration.
  • Multiple bookmark shelves.
  • Ability to save page fragments and entire PDF-documents.
  • Cloud bookmarking.
  • TODO functionality.
  • Support of Firefox Multi-Account containers.
  • Import/export from org-mode files and HTML-bookmarks.
  • Import of ScrapBook RDF archives.
  • It is possible to attach text notes to any bookmark or archived page.
  • Sharing to Pocket or Dropbox.
  • iShell integration.
  • Automation

Discover possible Scrapyard usage scenarios

It is possible to share links into the special Scrapyard Cloud shelf from mobile devices and get them on PC. The content of this shelf could also be browsed on your phone, in the case if you added bookmarks there from your computer. By using the Cloud shelf or sharing to other applications, you can have the necessary bookmarks or archived pages available on all your devices.

Android application

Scrapyard for Android allows to share links and text to Scrapyard Cloud shelf from any app on mobile devices.

Limitations

  • Text shared from the Android platform does not include links or images.
  • Selected text is shared as notes without a source URL. Share text from the Pocket app to save source URLs automatically.
