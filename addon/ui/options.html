Scrapyard Helper Application

Scrapyard helper application allows to transcend the limits of Firefox WebExtensions by providing several features that are otherwise impossible:

  • Import and browse Scrapbook RDF archives from any filesystem location.
  • Run scripts and display embedded objects at the saved pages.
  • Improve export performance.
  • Upload local files.
  • Manage backups.

On Windows the helper application could be installed automatically without the need of any configuration. It could also be installed manually on any system that supports Python 3.6+ with Tcl/Tk (you may need to consult Mozilla Wiki for more details on the configuration of native messaging).
