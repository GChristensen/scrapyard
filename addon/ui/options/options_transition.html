Transferring Scrapyard Content to the Filesystem

Since version 2, Scrapyard stores archived content in the filesystem. This makes the installation of the Scrapyard helper application mandatory. If you are not ready for this change, please install Scrapyard version 1 by this link and disable automatic updates in the add-on settings at about:addons (Firefox only, rollback to version 1 on Chrome is impossible).

The content archived with the Scrapyard version 1 is stored in the browser internal storage and is not directly accessible by version 2 of the add-on. The instructions below describe the process of transferring the archived content from the browser internal storage to the filesystem.

  1. Before performing the transition, you might want to back up your browser profile. In Firefox, you can find your browser profile path at the about:support settings page. In Chrome the current profile path could be found by the following URL: chrome://version
  2. Close your browser and install the Scrapyard helper application version 2.0+.
  3. Open the Settings Scrapyard options tab and specify the directory where you want to keep your archived content.
  4. Click on the "Transfer content" link.
  5. Make sure that the archived content is transferred successfully.
  6. Open the Advanced Scrapyard options tab and click on the "Reset internal storage" link.
