Scrapyard Helper Application

Scrapyard helper application allows to transcend the limits of WebExtensions by providing access to the filesystem and several other features that are otherwise impossible to achieve.

On Windows, the helper application could be installed automatically without the need for any configuration. The CLI-based installer runs on any OS but requires Python 3.7+.

It is necessary to restart the browser after the application has been installed. Please close your browser completely before updating to a new version.
